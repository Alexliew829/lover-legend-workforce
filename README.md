# Lover Legend Workforce ERP V1.74

Mobile-first workforce and payroll system for Lover Legend.

## Current modules
- 工人管理
- 扣款管理
- Payroll
- Payslip（开发中）
- Dashboard（开发中）

## Format standards
- 日期：dd-MM-yyyy
- 金额：1,000.00
- 手机优先、短字段双栏显示

## Important
Do not overwrite `js/config.js` after setting your Apps Script Web App URL.
Use `js/config.example.js` as template.
